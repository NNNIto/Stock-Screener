【A】トレンドフォロー系（順張り）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ A-1: 王道モメンタム戦略
  難易度: ★★☆  期待保有期間: 1〜3ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 株価 > 25日MA > 75日MA > 200日MA（完全順列）
  2. ゴールデンクロス発生（25日MA が 75日MA を上抜け）
  3. 出来高急増（平均出来高の1.5倍以上）
  4. RSI 50〜70（過熱しすぎず上昇中）
  [フィルター]
  - 時価総額 300億円以上
  - 1日売買代金 3億円以上
  [想定根拠]
  トレンド・テクニカル・出来高の三重確認。偽ブレイクを排除。

■ A-2: ブレイクアウト + 出来高確認
  難易度: ★★☆  期待保有期間: 2週間〜2ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 52週高値を終値でブレイクアウト
  2. ブレイク日の出来高 > 過去20日平均の2倍
  3. ADX > 25（トレンド強度確認）
  4. 株価 > 200日MA
  [フィルター]
  - 時価総額 100億円以上
  - 直近決算で増収増益
  [想定根拠]
  タートル・トレーディング系。大量出来高のブレイクは機関投資家参入のサイン。

■ A-3: MACD + RSI トレンド追従
  難易度: ★☆☆  期待保有期間: 1〜2ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. MACDラインがシグナルラインを上抜け（ゼロライン上で）
  2. MACDヒストグラムがプラス転換
  3. RSI 45〜65（モメンタム上昇中、過熱前）
  4. 株価 > 75日MA
  [フィルター]
  - 時価総額 200億円以上
  [想定根拠]
  2つの独立した指標が同時に強気シグナルを出した場合の信頼性向上。

■ A-4: 一目均衡表 三役好転
  難易度: ★★★  期待保有期間: 1〜3ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 転換線 > 基準線（短期上昇）
  2. 株価 > 雲の上限（中期上昇）
  3. 遅行線 > 26日前の株価（長期上昇）
  4. 雲が上昇形（先行スパン1 > 先行スパン2）
  [フィルター]
  - 時価総額 500億円以上（大型株推奨）
  [想定根拠]
  日本株特有の指標。機関投資家も参照。三役揃うと強力なトレンド確認。

■ A-5: 相対強度 + ADX 上昇相場乗り
  難易度: ★★☆  期待保有期間: 2〜3ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 過去3ヶ月リターンが TOPIX を10%以上上回る
  2. ADX > 30 かつ +DI > -DI
  3. 株価 > 200日MA
  4. ボリンジャーバンド上半分に位置
  [フィルター]
  - 時価総額 200億円以上
  - 1日売買代金 2億円以上
  [想定根拠]
  強い株はより強くなる（モメンタム効果）。市場全体と比較して強さを確認。

【B】バリュー + カタリスト系（割安株反発）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ B-1: グレアム流 低PBR + 高ROE
  難易度: ★★☆  期待保有期間: 2〜6ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. PBR < 1.0（純資産割れ）
  2. ROE > 10%（収益力確認）
  3. 自己資本比率 > 40%（財務健全）
  4. 株価が75日MAより上（トレンド確認）
  [フィルター]
  - 時価総額 100〜3000億円
  - 連続黒字3期以上
  [想定根拠]
  割安だが収益力がある銘柄。PBR改善要求（東証プレッシャー）追い風。

■ B-2: 低PER + 増益 + テクニカル確認
  難易度: ★★☆  期待保有期間: 1〜3ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. PER < セクター平均 × 0.7（割安水準）
  2. EPS成長率 > 15%（前年同期比）
  3. 業績上方修正直後（1ヶ月以内）
  4. MACD 買いシグナル発生
  [フィルター]
  - 時価総額 300億円以上
  - アナリストカバレッジ あり
  [想定根拠]
  業績上方修正は株価上昇の強力なカタリスト。割安×成長の組み合わせ。

■ B-3: 高配当 + 低負債 + テクニカル底打ち
  難易度: ★☆☆  期待保有期間: 2〜6ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 配当利回り > 3.5%
  2. 配当性向 < 60%（減配リスク低）
  3. 有利子負債/自己資本 < 0.5
  4. RSI < 35 からの反発（底打ち確認）
  5. 増配継続3年以上
  [フィルター]
  - 時価総額 500億円以上（安定銘柄）
  [想定根拠]
  配当収入を確保しながら株価回復を狙う。下値が配当利回りで支えられる。

■ B-4: 低PSR + 高成長 グロースバリュー
  難易度: ★★★  期待保有期間: 3〜6ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. PSR < 2.0（売上高対比で割安）
  2. 売上高成長率 > 20%（高成長）
  3. 営業利益率 > 10%（収益化済み）
  4. 株価がボリンジャーバンド下限から反発
  [フィルター]
  - 時価総額 50〜500億円（中小型成長株）
  - 上場3年以内の新興株は除外
  [想定根拠]
  成長株を割安なうちに仕込む。PSRは成熟度低い成長株に有効。

【C】逆張り・リバーサル系
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ C-1: 過売圏 ファンダ良好 反発狙い
  難易度: ★★☆  期待保有期間: 2週間〜2ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. RSI < 30（売られすぎ）
  2. 株価 > 200日MA（長期上昇トレンド内の押し目）
  3. ROE > 12%（業績裏付け）
  4. 直近決算 増収増益
  5. 株価が52週安値から10%以上上（底割れリスク低減）
  [フィルター]
  - 時価総額 300億円以上
  [想定根拠]
  ファンダ良好な株が一時的に売られすぎた局面を拾う。長期MAが下値支持。

■ C-2: ボリンジャー下限タッチ + ストキャスティクス
  難易度: ★★☆  期待保有期間: 2週間〜1ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 終値がボリンジャーバンド -2σ タッチまたは下抜け後の戻し
  2. スロー・ストキャスティクス %K < 20 から上向き転換
  3. 出来高が前日比 減少（売り圧力の衰え）
  4. 株価 > 75日MA（中期トレンド維持）
  [フィルター]
  - 時価総額 200億円以上
  [想定根拠]
  統計的に価格が平均回帰しやすい局面。出来高減少で売り枯れ確認。

■ C-3: 52週安値圏 + 需給改善
  難易度: ★★★  期待保有期間: 1〜3ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 52週安値から 5%以内（深押し）
  2. 信用倍率の低下傾向（需給改善）
  3. 空売り比率の急低下（売り方撤退）
  4. PBR < 0.8（資産価値割れ）
  5. 営業キャッシュフロー > 0
  [フィルター]
  - 時価総額 100〜500億円
  - 赤字企業は除外
  [想定根拠]
  バリュートラップ回避のため需給改善を必須条件に。売り方の撤退がトリガー。

■ C-4: MACDダイバージェンス 底打ち
  難易度: ★★★  期待保有期間: 1〜2ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 株価が安値を更新したがMACDヒストグラムは前回安値より高い（強気ダイバージェンス）
  2. RSI も同様のダイバージェンス形成
  3. 出来高が直近安値時より少ない（売り圧力低下）
  4. ファンダメンタルに大きな問題なし（赤字でない）
  [フィルター]
  - 時価総額 200億円以上
  [想定根拠]
  価格と指標の乖離は売り勢力の枯渇サイン。2指標同時で信頼性向上。

【D】CANSLIM系（成長株モメンタム）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ D-1: CANSLIM 完全版
  難易度: ★★★  期待保有期間: 1〜6ヶ月
  ─────────────────────────────
  [エントリー条件]
  C: 直近四半期EPS成長率 > 25%（前年同期比）
  A: 年間EPS成長率 > 25%（3年平均）
  N: 新製品・新サービス・新経営体制など カタリスト あり
  S: 出来高急増を伴う株価ブレイクアウト
  L: 業種内でトップ3の相対強度
  I: 機関投資家の保有増加トレンド
  M: 市場全体が上昇トレンド（TOPIX が200日MA上）
  [フィルター]
  - 時価総額 50〜2000億円
  - 株価 500円以上
  [想定根拠]
  ウィリアム・オニール考案。成長株の特性を7要素で網羅確認。

■ D-2: CANSLIM簡易版（実践向け）
  難易度: ★★☆  期待保有期間: 1〜3ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 直近四半期EPS成長率 > 20%
  2. 売上高成長率 > 15%
  3. 52週高値ブレイクアウト（出来高2倍以上）
  4. 業種内相対強度 上位20%
  5. ROE > 17%
  [フィルター]
  - 時価総額 100〜3000億円
  - 市場全体 上昇トレンド時のみ
  [想定根拠]
  CANSLIM の核心要素のみ抽出。スクリーニングで実装しやすい形に簡略化。

【E】需給・テーマ系
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ E-1: 自社株買い + テクニカル確認
  難易度: ★☆☆  期待保有期間: 1〜3ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 自社株買い実施発表（1ヶ月以内）
  2. 取得規模 > 発行済株式数の2%
  3. 株価 > 25日MA
  4. MACD 買いシグナル
  [フィルター]
  - 時価総額 300億円以上
  - 自己資本比率 > 40%（資金余力確認）
  [想定根拠]
  自社株買いは需給改善の直接的アクション。経営陣の自社株割安認識のサイン。

■ E-2: 業績上方修正 + ブレイクアウト
  難易度: ★★☆  期待保有期間: 2週間〜2ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 業績上方修正発表（2週間以内）
  2. 修正幅 > 経常利益 10%以上の上方修正
  3. 修正後の上昇で新高値ブレイク
  4. 出来高急増（平均の2倍以上）
  [フィルター]
  - 時価総額 200億円以上
  - 修正後もPER < 25倍
  [想定根拠]
  上方修正後の株価上昇は短期〜中期で持続しやすいアノマリー。

■ E-3: セクターローテーション + 個別強度
  難易度: ★★★  期待保有期間: 1〜3ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 対象セクターの過去1ヶ月相対強度がTOPIX比 +5%以上
  2. セクター内で個別銘柄の相対強度が上位10%
  3. 株価 > 200日MA
  4. RSI 50〜70
  [フィルター]
  - 時価総額 300億円以上
  [想定根拠]
  上昇セクターの中からさらに強い銘柄を選別。二重の強度フィルター。

■ E-4: 空売り比率急低下 + 底打ち確認
  難易度: ★★★  期待保有期間: 2週間〜1ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 空売り比率が過去1ヶ月で30%以上低下
  2. 信用倍率の改善（売り残減少）
  3. RSI < 40 からの反発
  4. 株価が直近安値を割らず下値固め
  [フィルター]
  - 時価総額 100億円以上
  - 赤字企業除外
  [想定根拠]
  売り方の撤退が買い圧力に変わる局面を捉える。需給転換の初動を狙う。

■ F-1: 高品質 コア保有銘柄スクリーニング
  難易度: ★☆☆  期待保有期間: 3〜6ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. ROE > 15% を3期連続維持
  2. 自己資本比率 > 50%
  3. 営業キャッシュフロー > 純利益（キャッシュ創出力）
  4. 5年連続増配
  5. 株価 > 200日MA かつ 乖離率 < +10%（過熱なし）
  [フィルター]
  - 時価総額 1000億円以上（大型安定株）
  [想定根拠]
  バフェット型の高品質株。長期保有でも機能するが中期でも有効。

■ F-2: 低ボラティリティ + 安定配当
  難易度: ★☆☆  期待保有期間: 2〜6ヶ月
  ─────────────────────────────
  [エントリー条件]
  1. 過去6ヶ月の価格変動率（σ）が市場平均の70%以下
  2. 配当利回り > 3%
  3. 配当性向 40〜60%（安定的）
  4. 株価 > 75日MA
  5. 自己資本比率 > 45%
  [フィルター]
  - 時価総額 500億円以上
  [想定根拠]
  リスク抑制しながらインカムゲインを確保。守りの中期戦略。
