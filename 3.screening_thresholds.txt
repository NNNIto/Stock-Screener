【A】トレンドフォロー系
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ A-1: 王道モメンタム戦略
  ─────────────────────────────
  [テクニカル]
  [必] 株価 > 25日SMA
  [必] 25日SMA > 75日SMA
  [必] 75日SMA > 200日SMA           ← 完全順列
  [必] ゴールデンクロス発生日 <= 20営業日以内
  [必] 出来高(当日) >= 20日平均出来高 × 1.5倍
  [必] RSI(14) >= 50 かつ <= 70
  [推] 株価の25日SMAからの乖離率 <= +15%  ← 過熱回避
  [除] RSI(14) > 75                  ← 過熱域除外
  [フィルター]
  [必] 時価総額 >= 300億円
  [必] 20日平均売買代金 >= 3億円
  [必] 株価 >= 300円
  [除] 決算発表 ±5営業日

■ A-2: ブレイクアウト + 出来高確認
  ─────────────────────────────
  [テクニカル]
  [必] 終値 >= 52週高値（前日比較でブレイク当日）
  [必] ブレイク日出来高 >= 20日平均出来高 × 2.0倍
  [必] ADX(14) >= 25
  [必] 株価 > 200日SMA
  [推] ATR(14) 拡大（前5日平均ATR比 >= 1.2倍）
  [除] 52週高値が直近5日以内の場合（連日高値は対象外）
  [ファンダ]
  [必] 直近決算 売上高 前年同期比 >= 0%（増収）
  [必] 直近決算 営業利益 前年同期比 >= 0%（増益）
  [フィルター]
  [必] 時価総額 >= 100億円
  [必] 20日平均売買代金 >= 1億円

■ A-3: MACD + RSI トレンド追従
  ─────────────────────────────
  [テクニカル]
  [必] MACD(12,26,9) ラインがシグナルラインを上抜け（当日または前1日以内）
  [必] MACDライン > 0（ゼロライン上でのクロス）
  [必] MACDヒストグラム > 0 かつ 前日比でプラス拡大
  [必] RSI(14) >= 45 かつ <= 65
  [必] 株価 > 75日SMA
  [推] 株価の75日SMAからの乖離率 <= +12%
  [フィルター]
  [必] 時価総額 >= 200億円
  [必] 20日平均売買代金 >= 1億円

■ A-4: 一目均衡表 三役好転
  ─────────────────────────────
  [テクニカル]
  [必] 転換線(9日) > 基準線(26日)
  [必] 終値 > 先行スパン1（雲の上限）
  [必] 終値 > 先行スパン2（雲の下限）
  [必] 遅行線 > 26日前の終値
  [必] 先行スパン1 > 先行スパン2（雲が陽雲）
  [推] 雲の厚み >= 株価の2%（薄い雲は信頼性低）
  [推] 転換線と基準線の乖離 >= 0.5%
  [除] 雲のねじれ（スパン1とスパン2の交差）が直近5日以内
  [フィルター]
  [必] 時価総額 >= 500億円
  [必] 20日平均売買代金 >= 5億円

■ A-5: 相対強度 + ADX 上昇相場乗り
  ─────────────────────────────
  [テクニカル]
  [必] 過去60営業日リターン >= TOPIX 同期間リターン + 10%
  [必] ADX(14) >= 30
  [必] +DI(14) > -DI(14)
  [必] 株価 > 200日SMA
  [必] ボリンジャーバンド(20,2σ): 終値 >= 中心線（20日SMA）
  [推] 過去20営業日リターン >= TOPIX 同期間 + 5%
  [フィルター]
  [必] 時価総額 >= 200億円
  [必] 20日平均売買代金 >= 2億円
  [必] TOPIX が 200日SMA 上（上昇相場確認）

【B】バリュー + カタリスト系
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ B-1: グレアム流 低PBR + 高ROE
  ─────────────────────────────
  [ファンダ]
  [必] PBR <= 1.0倍
  [必] ROE(直近本決算) >= 10%
  [必] 自己資本比率 >= 40%
  [必] 連続黒字 >= 3期
  [推] PBR <= 0.8倍（より割安）
  [推] ROE >= 12%
  [除] 有利子負債/自己資本 > 2.0倍
  [除] 直近1期で最終赤字
  [テクニカル]
  [必] 株価 > 75日SMA
  [推] RSI(14) >= 40（底割れ傾向でない）
  [フィルター]
  [必] 時価総額 >= 100億円 かつ <= 3000億円
  [必] 20日平均売買代金 >= 0.5億円

■ B-2: 低PER + 増益 + テクニカル確認
  ─────────────────────────────
  [ファンダ]
  [必] PER <= 業種平均PER × 0.7倍
  [必] PER >= 5倍（赤字・極端割安除外）
  [必] 直近四半期EPS 前年同期比 >= 15%
  [必] 業績上方修正 発表日 <= 20営業日以内
  [必] 上方修正幅(経常利益) >= 10%
  [推] PER <= 15倍（絶対水準でも割安）
  [除] PER < 5倍（異常値除外）
  [テクニカル]
  [必] MACD(12,26,9) 買いシグナル（クロス発生 <= 10日以内）
  [推] RSI(14) >= 45
  [フィルター]
  [必] 時価総額 >= 300億円
  [必] アナリスト予想カバー >= 1社

■ B-3: 高配当 + 低負債 + テクニカル底打ち
  ─────────────────────────────
  [ファンダ]
  [必] 配当利回り >= 3.5%
  [必] 配当性向 >= 20% かつ <= 60%
  [必] 有利子負債/自己資本(D/Eレシオ) <= 0.5倍
  [必] 増配継続年数 >= 3年
  [推] 配当利回り >= 4.0%
  [推] 増配継続 >= 5年
  [除] 配当性向 > 80%（減配リスク高）
  [除] 直近1期で最終赤字
  [テクニカル]
  [必] RSI(14) <= 35 からの上向き転換（直近5日以内）
  [推] 株価 > 200日SMA（長期トレンド維持中の押し目）
  [フィルター]
  [必] 時価総額 >= 500億円
  [必] 20日平均売買代金 >= 1億円

■ B-4: 低PSR + 高成長 グロースバリュー
  ─────────────────────────────
  [ファンダ]
  [必] PSR(株価売上高倍率) <= 2.0倍
  [必] 売上高成長率(前年同期比) >= 20%
  [必] 営業利益率 >= 10%
  [必] 売上高成長率 直近2四半期 連続 >= 15%
  [推] PSR <= 1.5倍
  [推] 営業利益率 >= 15%
  [除] 直近1期で営業赤字
  [除] 上場3年未満（初値バブル除外）
  [テクニカル]
  [必] ボリンジャーバンド(20,2σ) 下限から反発（終値が -1σ 以上への戻し）
  [推] RSI(14) >= 35（底割れしていない）
  [フィルター]
  [必] 時価総額 >= 50億円 かつ <= 500億円
  [必] 20日平均売買代金 >= 0.3億円

【C】逆張り・リバーサル系
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ C-1: 過売圏 ファンダ良好 反発狙い
  ─────────────────────────────
  [テクニカル]
  [必] RSI(14) <= 30（売られすぎ域）
  [必] 株価 > 200日SMA（長期トレンド上の押し目）
  [必] 終値 >= 52週安値 × 1.10（10%以上 52週安値より上）
  [推] RSI(14) が 直近3日で上向き転換
  [推] 株価が 200日SMA から -5% 〜 -15% 乖離（深押し確認）
  [除] 株価 < 200日SMA（トレンド崩壊の可能性）
  [ファンダ]
  [必] ROE(直近本決算) >= 12%
  [必] 直近決算 増収増益（売上・営業利益ともに前年同期比 >= 0%）
  [必] 連続黒字 >= 2期
  [除] 直近1期で最終赤字
  [フィルター]
  [必] 時価総額 >= 300億円
  [必] 20日平均売買代金 >= 1億円

■ C-2: ボリンジャー下限タッチ + ストキャスティクス
  ─────────────────────────────
  [テクニカル]
  [必] 直近3日以内に終値がBB(20,2σ) -2σ 以下をタッチ
  [必] 現在の終値 > BB -2σ（タッチ後の戻し確認）
  [必] スロー・ストキャスティクス(14,3): %K <= 20 かつ %K が %D を上抜け
  [必] 当日出来高 <= 5日平均出来高 × 0.8倍（出来高減少=売り枯れ）
  [必] 株価 > 75日SMA
  [推] BB バンド幅(BandWidth) が収縮傾向（スクイーズ明け）
  [推] RSI(14) <= 35
  [除] BB -2σ を連続3日以上下回っている（トレンド転落の可能性）
  [フィルター]
  [必] 時価総額 >= 200億円
  [必] 20日平均売買代金 >= 1億円

■ C-3: 52週安値圏 + 需給改善
  ─────────────────────────────
  [テクニカル・需給]
  [必] 終値 <= 52週安値 × 1.05（5%以内の安値圏）
  [必] 信用倍率(信用買い残/信用売り残): 直近4週で低下傾向
  [必] 空売り比率: 直近4週で -10%以上低下（例 40% → 30%）
  [必] 株価が直近20日の最安値を更新していない（下値固め確認）
  [推] 信用売り残 前週比 -5%以上の減少
  [ファンダ]
  [必] PBR <= 0.8倍
  [必] 営業キャッシュフロー > 0（直近本決算）
  [必] 最終黒字（直近本決算）
  [除] 直近1期で最終赤字
  [除] 上方修正なし かつ 下方修正あり（直近6ヶ月以内）
  [フィルター]
  [必] 時価総額 >= 100億円 かつ <= 500億円
  [必] 20日平均売買代金 >= 0.5億円

■ C-4: MACDダイバージェンス 底打ち
  ─────────────────────────────
  [テクニカル]
  [必] 直近2つの安値を比較:
       株価安値2 < 株価安値1（価格は安値更新）
       MACDヒストグラム安値2 > MACDヒストグラム安値1（指標は逆行）
       ← 強気ダイバージェンス
  [必] RSI(14)でも同様のダイバージェンス:
       RSI安値2 > RSI安値1（株価安値更新時にRSIは前安値より高い）
  [必] 直近安値日の出来高 < 前回安値日の出来高 × 0.8倍
  [必] 2つの安値の間隔 >= 10営業日（短すぎるダイバージェンスは除外）
  [推] MACDラインがシグナルラインに接近または上抜け
  [ファンダ]
  [必] 直近本決算 最終黒字
  [必] 自己資本比率 >= 30%
  [除] 直近1期で最終赤字
  [フィルター]
  [必] 時価総額 >= 200億円
  [必] 20日平均売買代金 >= 0.5億円

■ D-1: CANSLIM 完全版
  ─────────────────────────────
  C [Current Earnings]
  [必] 直近四半期EPS 前年同期比 >= 25%
  [推] >= 40%（より強力）
  [除] EPS マイナス（赤字四半期）

  A [Annual Earnings]
  [必] 直近3期平均 年間EPS成長率 >= 25%
  [必] 直近3期 連続EPS増加
  [推] 直近本決算 ROE >= 17%

  N [New Product/Service/Management]
  [必] 以下のいずれか1つ以上:
       - 新製品・新サービス 直近6ヶ月以内の発表
       - 新規大口契約・提携 直近3ヶ月以内
       - 経営体制刷新（新CEO等）直近6ヶ月以内

  S [Supply & Demand]
  [必] 終値が直近52週高値を更新（ブレイクアウト当日）
  [必] ブレイク日出来高 >= 50日平均出来高 × 2.0倍
  [推] 出来高 >= 50日平均 × 3.0倍（より強力）

  L [Leader or Laggard]
  [必] 業種内の相対強度ランキング 上位20%以内
  [推] 上位10%以内
  [除] 業種内ランキング 下位50%

  I [Institutional Sponsorship]
  [必] 機関投資家保有比率 直近四半期比較で増加
  [必] 機関投資家保有比率 >= 10%
  [推] 機関投資家数 前四半期比 増加

  M [Market Direction]
  [必] TOPIX が 200日SMA 上
  [必] TOPIX の直近20日リターン >= -3%
  [推] TOPIX が 25日SMA 上かつ 上昇トレンド

  [フィルター]
  [必] 時価総額 >= 50億円 かつ <= 2000億円
  [必] 株価 >= 500円
  [必] 20日平均売買代金 >= 0.5億円

■ D-2: CANSLIM簡易版（実践向け）
  ─────────────────────────────
  [ファンダ]
  [必] 直近四半期EPS 前年同期比 >= 20%
  [必] 売上高成長率(前年同期比) >= 15%
  [必] ROE(直近本決算) >= 17%
  [必] 連続黒字 >= 2期

  [テクニカル]
  [必] 終値 >= 52週高値（ブレイクアウト当日）
  [必] ブレイク日出来高 >= 20日平均出来高 × 2.0倍

  [相対強度]
  [必] 過去60営業日の株価リターン: 市場全体の上位20%以内
  [推] 上位10%以内

  [市場環境]
  [必] TOPIX >= 200日SMA

  [フィルター]
  [必] 時価総額 >= 100億円 かつ <= 3000億円
  [必] 20日平均売買代金 >= 1億円

■ E-1: 自社株買い + テクニカル確認

  [除] 自社株買い発表後すでに +10%以上上昇済み

  [テクニカル]

■ E-2: 業績上方修正 + ブレイクアウト
  ─────────────────────────────
  [ファンダ]
  [必] 業績上方修正 発表日 <= 10営業日以内
  [必] 上方修正幅(経常利益) >= 10%
  [推] 上方修正幅 >= 20%
  [必] 修正後PER <= 25倍

  [テクニカル]
  [必] 修正発表後の株価が新高値をブレイク
  [必] ブレイク日出来高 >= 20日平均出来高 × 2.0倍
  [推] 株価 > 200日SMA

  [フィルター]
  [必] 時価総額 >= 200億円
  [必] 20日平均売買代金 >= 1億円

■ E-3: セクターローテーション + 個別強度
  ─────────────────────────────
  [セクター条件]
  [必] 対象セクターの過去20営業日リターン >= TOPIX 同期間 + 5%
  [必] 対象セクターが TOPIX 比で3ヶ月で上位3セクター以内
  [推] セクター内銘柄の平均RSIが上昇トレンド

  [個別銘柄]
  [必] 業種内 相対強度ランキング 上位10%以内
  [必] 株価 > 200日SMA
  [必] RSI(14) >= 50 かつ <= 70
  [推] ADX(14) >= 25

  [フィルター]
  [必] 時価総額 >= 300億円
  [必] 20日平均売買代金 >= 2億円

■ E-4: 空売り比率急低下 + 底打ち確認
  ─────────────────────────────
  [需給]
  [必] 空売り比率 直近20営業日で >= 30%の低下率
       (例: 40% → 28%以下 = 30%低下)
  [必] 信用売り残 直近4週で低下傾向
  [必] 信用倍率(買い残/売り残) 直近4週で上昇傾向
  [推] 空売り比率の絶対値 <= 25%（現在値が低い）

  [テクニカル]
  [必] RSI(14) 直近5日以内に <= 40 をつけた後、現在 >= 40
  [必] 直近20日間の最安値 更新なし（下値固め確認）
  [推] 株価 > 75日SMA（中期トレンド維持）

  [ファンダ]
  [必] 最終黒字（直近本決算）
  [除] 直近1期で最終赤字

  [フィルター]
  [必] 時価総額 >= 100億円
  [必] 20日平均売買代金 >= 0.5億円

【F】品質・安定成長系
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ F-1: 高品質 コア保有銘柄
  ─────────────────────────────
  [ファンダ - 収益性]
  [必] ROE >= 15%（直近3期連続）
  [必] 自己資本比率 >= 50%
  [必] 営業キャッシュフロー >= 当期純利益 × 0.8倍
  [必] フリーキャッシュフロー > 0

  [ファンダ - 株主還元]
  [必] 増配継続 >= 5年
